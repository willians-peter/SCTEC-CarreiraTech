# SCTEC-CarreiraTech

Landing page funcional destinada à apresentação de um portfólio pessoal
Peter Willian's Cipriano

## 1. Descrição do Projeto

Esta landing page foi desenvolvida como projeto final para o curso de programação do SENAI/SC LAB365. Com objetivo de consolidar minha trajetória
profissional, unindo minha experiência na área da saúde e buscando adquirir conhecimento e pratica em tecnoloias de softwares.

## 2. Tecnologias Utilizadas

Para atingir o nível máximo de exigência técnica, o projeto utiliza:

- **HTML5:** Estruturação semântica de múltiplas páginas.
- **CSS3:** Estilização moderna, uso de Flexbox para alinhamento e variáveis CSS para suporte a temas.
- **JavaScript:** Implementação de lógica de saudação dinâmica (Bom dia/tarde/noite), relógio em tempo real e manipulador de eventos para o Dark Mode.


## 3. Funcionalidades

- **Navegação Modular:** Menu fixo em todas as páginas para facilitar o acesso.
- **Sistema de Temas:** Botão que alterna entre Dark Mode e Light Mode, persistindo a preferência visual.
- **Seção de Projetos:** Exibição de aplicações reais, como a Calculadora de Insulina desenvolvida para uso em ambiente de UTI,
  com links diretos para demonstração e código-fonte no GitHub.
- **Responsividade:** Layout adaptável para diferentes tamanhos de tela.

## 4. Habilidades Técnicas e Comportamentais

- **Hard Skills:** Java, Spring Boot, React, MySQL, HTML, CSS e JavaScript.
- **Soft Skills:** Resiliência (atuação na linha de frente COVID-19), rigor com protocolos de segurança, manejo de riscos críticos e aprendizado contínuo.

## 5. Como executar o projeto

1. Extraia o arquivo .zip enviado.
2. Abra o arquivo `index.html` em qualquer navegador moderno.
3. Navegue pelas seções através do menu superior.
4. Acesso pelo link: (projeto disponivel no github)
